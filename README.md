# michele-colombo.github.io

Personal portfolio website showcasing my projects and background.

## Sections

- **About Me**: Personal introduction and background (to be customized)
- **Projects**: Showcase of public GitHub repositories including:
  - PC-Gau: Shape Matching research project (MATLAB)
  - Adrenaline Board Game: Software Engineering project (Java)
  - ESP32 Gimbal: Hardware control project (ESP32)
- **Contact**: Links to GitHub, email, and LinkedIn

## Customization

To customize this website:

1. Update the "About Me" section in `index.html` (replace the placeholder text)
2. Add project images to the `images/` directory:
   - `pc-gau-placeholder.png`
   - `adrenaline-placeholder.png`
   - `gimbal-placeholder.png`
3. Update contact links in the Contact section (email and LinkedIn)

## Local Development

Simply open `index.html` in a web browser to view the website locally.

## Deployment

This website is designed to be hosted on GitHub Pages. Once pushed to the `main` branch, it will be automatically available at `https://michele-colombo.github.io`.
